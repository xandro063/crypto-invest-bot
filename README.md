# Crypto Invest Bot

Telegram bot for automatic crypto trading with 25-35% monthly profit.

## Quick Start

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file with your bot token
4. Run: `python main.py`

## Features

- Two balance system (available/trading)
- 2-level referral (10% + 5%)
- Admin panel for management
- Transaction history
- FAQ support
